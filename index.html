Training Manual

Verify each ship’s documents and decide to APPROVE, DENY, or DETAIN. Use the scanner and rules below.

  1. Names present & legible on Passport, Visa, Photo ID and must match exactly. Smudged Visa name fails.
  2. Passport unexpired compared to today (shown at top).
  3. Visa seal valid: NX-### only. FORGE or any other format is invalid.
  4. Genetic Key clean. Use Scan Genetic Key or type a GK to check. If wanted, DETAIN.
  5. Photo ID matches live portrait features (eyes, arms, antennae, head, markings).

Shortcuts: A Approve, D Deny, F Detain, N Next, Space Start/End Day.
